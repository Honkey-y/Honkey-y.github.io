问题：不管是yarn、npm、cnpm都无法卸载vue2.9.6

今日在巩固Vue知识，做demo时打算换做vue3.x版本，却发现卸载全局vue-cli时一直失效

>  npm uninstall vue-cli

up to date, audited 1 package in 328ms
found 0 vulnerabilities

>  vue -V
2.9.6

如上，我在网上搜索了很多资料，甚至删除了c盘内的vue-cli文件，但发现都无用。

解决方法

  1. 删除 .npmc 文件，位置在 C:\Users\电脑用户名
  2. 执行 npm uninstall vue-cli -g
  3. 这时再次执行 vue-V，会发现版本号消失了
  4. npm install -g @vue/cli
  5. （若提示npm ERR code ERR_SOCKET_TIMEOUT npm ERR）执行 npm install rxjs，再次执行第四步
  6. 执行vue-V查看vue新版本是否安装成功
